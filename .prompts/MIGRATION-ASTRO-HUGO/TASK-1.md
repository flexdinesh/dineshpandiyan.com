## Objective
Install Hugo extended and create minimal `hugo.toml` configuration file.

## Requirements
- Use latest Hugo extended version (v0.152+)
- Static site generation only
- Enable Chroma syntax highlighting
- Configure sitemap auto-generation
- Setup RSS feed output

## Deliverables

### 1. Install Hugo Extended
Check if Hugo is installed, install if needed:
```bash
hugo version
```

### 2. Create `hugo.toml`
Create minimal config with:
- `baseURL` (check existing site URL from package.json or astro config)
- `title` (from existing config)
- `languageCode = 'en-us'`
- Enable Goldmark markdown processor
- Configure Chroma syntax highlighting
- Enable RSS feed
- Enable sitemap

### 3. Configuration Details

**Goldmark**: Default markdown processor, confirm it's enabled

**Chroma**: Configure syntax highlighting:
- Style preference (can use default)
- Line numbers if desired

**RSS**:
- Enable RSS output format
- Configure feed sections (blog, notes, talks, work)

**Sitemap**:
- Auto-generated by Hugo (confirm enabled in config)
